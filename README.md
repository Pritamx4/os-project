# Memory Management Simulator

A modern, interactive web-based simulator for visualizing memory management techniques including **Paging**, **Virtual Memory**, and **Segmentation**.

- **Paging Simulation**
  - FIFO (First In First Out) algorithm

  - Real-time statistics (Page Faults, Hits, Hit Rate)
  - Step-by-step execution timeline

- **Segmentation**
  - Segment table visualization
  - Address translation with validation
  - Physical memory map
  - Segmentation fault detection

- **Modern UI**

  - Professional typography (Inter & JetBrains Mono)
  - Smooth animations and transitions
  - Responsive design

### Paging Mode
1. Select your preferred algorithm (FIFO or LRU)
2. Set the number of frames
3. Enter a page reference string (comma-separated numbers)
4. Click "Run Simulation" to see the results
5. View the execution timeline and statistics

### Segmentation Mode
1. Switch to Segmentation mode
2. View the segment table
3. Enter segment number and offset
4. See the physical address translation or segmentation fault
